# 📱 Telegram QR Code Generator Bot

A fully functional, professional Telegram bot that generates QR codes with custom watermarks from any text or URL provided by the user. Includes a built-in "Force Join" system and an Admin Panel.

## ✨ Features
* **Instant QR Generation**: Converts text/URLs into high-quality QR codes.
* **Custom Watermarks**: Automatically adds your bot's username (or any text) to the bottom of the QR code.
* **Force Join System (Majburiy Obuna)**: Requires users to subscribe to specific channels before they can use the bot.
* **Admin Dashboard**:
* Broadcast messages to all users (`/broadcast`).
* Add/Remove mandatory channels on the fly (`/addchannel`, `/delchannel`).
* View user statistics (`/stats`).
* **SQLite Database**: Lightweight, built-in storage (no extra setup required).
